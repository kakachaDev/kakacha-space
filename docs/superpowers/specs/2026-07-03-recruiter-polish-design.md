# Перетряска сайта под «продажу» рекрутеру — дизайн

Дата: 2026-07-03. Согласовано с Егором в сессии (выборы: комментарии — Remark42/совместимый виджет за env-флагом, язык — русский, статы — гибрид фактов и шутки, посты — 4–5 с датами назад, меню — EN-названия + RU-подписи).

## Цель

Сайт читает рекрутер/тимлид с hh.ru. Всё должно продавать: максимум скилловых «понтов», но каждый — с обоснованием («что умею» → «чем докажу»). Никаких выдуманных цифр и процентов. Русский язык везде, игровая стилистика (pause menu) сохраняется.

## Ограничения (NDA)

Про текущую работу: только «высоконагруженный real-time PWA-продукт», «бизнес-логика», без названий компании/продукта, без «Telegram Mini App», «веб-игра», гемблинг-лексики. Проверяемые цифры: ~1000 коммитов / 22 месяца, 100+ компонентов из Figma, SDK на TS strict + Vitest.

## Разделы

### Character (Обо мне)
- Интро: 2 абзаца, кто такой и чем отличается (геймдев → фронтенд).
- «Статы»: проверяемые факты с подписью-объяснением у каждого (7+ лет; ~1000 коммитов — единственный фронтендер, 22 мес; 100+ компонентов из Figma; 3 учебных года преподавания). Плюс один шуточный StatBar (Кофе — 100) для тона.
- «Перки»: каждый перк = навык + строка-доказательство (движок частиц на Canvas 2D вместо vendor-либы; TS-SDK strict+тесты; оптимизация Lottie canvas/SVG; real-time WebSocket-синк; Paper-плагины и Godot в свободное время).
- Таймлайн 2019→сейчас (геймдев-джуниор → ведущий Unity → фронтенд соло).

### Quest Log (Проекты)
Расширить `Project`: `icon` (эмодзи), `tags` (технологии), `year`, `status` ('active'|'done'|'onhold'), `highlight` (строка «что здесь интересного»). Русские описания. Категории с русскими заголовками. Бейджи статуса в игровой лексике.

### Inventory (Стек)
Из «чем сделан сайт» → инвентарь навыков по категориям (Frontend / Backend и интеграции / Gamedev / Инструменты), каждый предмет — с пояснением, где применялся. Блок «из чего сделан этот сайт» остаётся внизу отдельно + ссылка на пост.

### Chronicle (Блог)
- Frontmatter: `tags` (через запятую), `cover` (путь к картинке в /blog/), существующие поля.
- Время чтения: `utils/readingTime.ts`, ~180 слов/мин, минимум 1 мин.
- Индекс: карточки — заголовок, дата (формат «24 мая 2026»), теги, время чтения, выдержка.
- Пост: мета-строка (дата · время чтения · теги), обложка, полноценные стили маркдауна (h2/h3, код, таблицы, цитаты, картинки).
- Комментарии: `CommentsSection.vue` — если задан `VITE_REMARK42_HOST`, встраивает Remark42; иначе если задан `VITE_GISCUS_REPO(+…)` — giscus; иначе не рендерится. Инструкция по развёртыванию — `docs/comments-setup.md`. Сервера сейчас нет — блок скрыт до настройки.
- +4 поста по реальному опыту (даты: февраль, март, апрель, июнь 2026): движок частиц Canvas 2D, Lottie canvas/SVG, TypeScript-SDK поверх REST/WS, преподавание. SVG-диаграммы в `public/blog/`. Существующим двум постам добавить теги.
